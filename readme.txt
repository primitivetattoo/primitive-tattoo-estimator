Interactive tattoo price estimator with promo banner support and Elementor widget.

== Description ==

A 4-step interactive price estimator for tattoo studios. Visitors select style, size, placement, and complexity to get an instant price range estimate, then book via WhatsApp.

Features:

* 4-step wizard: Style → Size → Placement → Complexity
* Dynamic pricing with configurable multipliers
* Promotional banner with toggle and custom URL
* WhatsApp integration with pre-filled messages
* Elementor widget with per-instance font overrides
* Custom heading and body font settings
* Auto-loads Google Fonts when needed
* Mobile responsive design
* IDR currency formatting

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate through the Plugins menu
3. Configure pricing and promo in Settings → Tattoo Estimator
4. Add `[ptb_estimator]` shortcode to any page, or use the Elementor widget
